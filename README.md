# Binary Trees

C++ practice repository for binary tree concepts and interview-style tree problems.

## Suggested Topics

- Tree node creation
- Preorder, inorder, postorder traversal
- Level-order traversal
- Binary search tree operations
- Tree height and depth
- Searching and insertion

## Build and Run

Add or compile a standalone `.cpp` file with:

```bash
g++ -std=c++17 "file.cpp" -o app
./app
```

## Status

This repository is currently a starter repository. Add tree implementations and keep each file focused on one concept or problem.
